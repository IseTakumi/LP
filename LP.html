WAO Let's Code

WAO Let's CodeはHTML,CSS,JavaScriptを中心としたプログラミング講座です。

WAO Let's Codeを学習することによってプログラミングの発想、考え方を得ることや
webページを実際に作ることができます。

映像での講座の受講によって、好きな時間、好きな場所で勉強することができます。

また、講座は単元ごとに細かく分かれて、膨大な演習量を含んでいるのでたくさんの知識を
得ることができます。

WAO Let's Codeが選ばれる理由

細分化されたカリキュラムと膨大な演習量

WAO Let's Codeでは48の単元、 数百の学習項目という細分化されたカリキュラム
それに対応したドリルという名の演習が膨大に含まれています。
これを完璧にすることによってプログラミング能力は格段に向上します。

WAO Let's Code受講者のコードと完成品が見れる

WAO Let's Codeは掲示板という機能があり、受講者であれば誰でも投稿でき、それを見ることができます。
たくさんの受講者のコードを見ることができるので、新たな知識や気付きを得られること間違いなし！

初心者にやさしい設定

WAO Let's Codeは他のプログラミング講座に比べて難しいパソコンの設定、使用環境の設定が一切ありません。
完全初心者の方でも、気軽に始めやすい講座になっています。

学習STEP

1

動画を見る

自分が学びたい単元を選び、その中に含まれている、1つ1～5分の動画を見ます。
実際に、動画の中の人がコードを書いているので新しいことでも安心して学ぶことができます。

2

コードを書く

STEP1の動画で学んだことを実際に自分で書いてみます。
コードの内容なども詳しく書いてあるので、コードを理解しながら書くことができます。

3

ドリルで演習をする

STEP1,2で学んだことを活かしてドリルで演習を行います。
その単元以前の内容も含まれることもあるので効率良くプログラミングを学ぶことができます。

HTML

webページを作成するために使用する言語

CSS

HTMLをどのように装飾するかを指定する

JavaScript

HTMLにどのように動きをつけるかを指定する
